# Function Definitions
## Character Classification
**`isalpha`** - Check if character is a letter (a-z, A-Z)
**`isdigit`** - Check if character is a digit (0-9)
**`isalnum`** - Check if character is alphanumeric (letter or digit)
**`isascii`** - Check if character is ASCII (0-127)
**`isprint`** - Check if character is printable (includes space)
## Character Conversion
**`toupper`** - Convert character to uppercase
**`tolower`** - Convert character to lowercase
## String Length
**`strlen`** - Get length of null-terminated string
## Memory Operations
**`memset`** - Fill memory block with a specific byte value
**`bzero`** - Fill memory block with zeros
**`memcpy`** - Copy memory block (source/dest cannot overlap)
**`memmove`** - Copy memory block (safe for overlapping regions)
**`memchr`** - Find byte in memory block
**`memcmp`** - Compare memory blocks byte by byte
## String Operations
**`strlcpy`** - Safely copy string with size limit
**`strlcat`** - Safely concatenate strings with size limit
**`strchr`** - Find first occurrence of character in string
**`strrchr`** - Find last occurrence of character in string
**`strncmp`** - Compare first n characters of two strings
**`strnstr`** - Find substring within length limit
## Conversion
**`atoi`** - Convert string to integer
